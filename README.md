# OpenWrt Backup & Restore Menu

Add a **Backup & Restore menu** for Nikki / Momo & SSClash in LuCI.

This script automatically installs a simple menu to manage backup and restore configurations from the LuCI interface.

---

## Features

* Backup configuration for Nikki / Momo
* Backup configuration for SSClash
* Restore configuration easily
* Simple installation script
* Lightweight and compatible with most OpenWrt builds

---

## Installation

Copy and paste this command in the OpenWrt terminal:

```sh
wget -O - https://raw.githubusercontent.com/Yogxx/BackupRestore/main/manage.sh | sh
```

---

## Compatibility

Tested on:

* OpenWrt 23.xx
* OpenWrt 24.xx
* OpenWrt 25.xx
* LuCI Web Interface

---

## Notes

* Make sure your device has internet access before running the script.
* The script will automatically create the required LuCI menu.
